# Local-Technician-Finder
MERN Stack, REST APIs, Full-Stack Development 
•	Built a full-stack platform for discovering verified local technicians with features like ratings, reviews, and location-based sorting.
•	Developed a responsive React frontend with service listings, booking forms, technician profiles, and an optional admin dashboard.
•	Implemented Node.js/Express backend with secure CRUD operations, slot management, and MongoDB integration for optimized data flow.
